# Sales Performance Analysis using Microsoft Excel

## Project Overview

This project analyzes a multi-year sales dataset to uncover revenue trends, profitability drivers, discount impact, and regional performance using Microsoft Excel.

The goal of this analysis was to transform raw transactional data into meaningful business insights through data cleaning, transformation, pivot tables, and an interactive KPI dashboard.

---

## Objective

- Analyze sales performance across years, regions, and product categories  
- Evaluate the impact of discounts on profit  
- Identify top revenue-generating products  
- Build an interactive and dynamic Excel dashboard  

---

## Tools & Techniques Used

- Microsoft Excel
- Data Cleaning & Transformation
- Excel Tables

- Pivot Charts
- Slicers (Interactive Filtering)

---

## Key Business Insights

- Revenue showed consistent growth across selected years.
- Higher discounts significantly reduce profit margins.
- A small percentage of products contribute to majority of revenue (Pareto effect).
- Certain regions outperform others in profitability.
- Some high-revenue products have low profit margins due to heavy discounting.
